# Build a booking event web application
## Technologies used in the project
 - Frontend: [ReactJS](https://reactjs.org)
 - Backend: [Spring Boot](https://spring.io/projects/spring-boot/)
 - Database: [MongoDB](https://www.mongodb.com)
